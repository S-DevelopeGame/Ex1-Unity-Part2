# Player With MiniMap

במשחק השחקן משחק דמות בנדית שלקוחה מליטל פייטר.\
השחקן יכול לנוע במפה ויש לו מיני מפה של המשחק.\
העולם שלו סגור יש גבולות למפה ואם אנסה לעבור אותם לא יצליח.\
השחקן יכול לנוע ימינה ושמאלה בלבד.\
